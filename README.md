# 花粉症・限界突破エクスキュース（言い訳）ジェネレーター
フロントエンド、バックエンドを含む全体をクローン

```bash
git clone --recursive https://github.com/toriaezu-yattemitai/excuse-by-pollen.git
```

## Dockerを用いて実行する
### ビルド＆起動
```bash
docker compose up --build
```

### 起動
```bash
docker compose up
```

### 終了
```bash
docker compose down
```

### バックグラウンド起動
```bash
docker compose up -d
```
